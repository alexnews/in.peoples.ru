## Design Principles

1. **Match existing site** â€” Same Bootstrap version, color scheme (accent red #d92228), fonts, layout grid
2. **Russian-first** â€” All UI text in Russian, left-aligned, proper Cyrillic typography
3. **Mobile-responsive** â€” All forms usable on phones (many Russian users browse on mobile)
4. **Progressive enhancement** â€” Forms work without JS, JS adds autocomplete/drag-drop/AJAX
